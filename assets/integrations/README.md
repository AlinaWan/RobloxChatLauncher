> [!WARNING]
> **Work In Progress: Integrations**  
>
> Do not use these integrations in a live game yet.
>
> The backend infrastructure is not yet set up to handle inbound or outbound data.  
> The client also doesn't know which user sent a command, so the server won't know
> which user is supposed to perform a requested action in the game.
>
> Attempting to use these integrations in a live game
> will result in connection failures or 404 errors until the backend infrastructure is deployed.
